# HougoBot

A simple dicord bot who play random music in a directory.

1. `npm install`

2. edit `bot.json`
```json
[
  {
    "email": "bot.email@.com",
    "password": "botpassword"
  },
  {
    "textChan": "music",
    "voiceChan": "Jukebox"
  },
  {
    "musicDir": "C:/Users/Public/Music"
  }
]
```

3. `npm start`
